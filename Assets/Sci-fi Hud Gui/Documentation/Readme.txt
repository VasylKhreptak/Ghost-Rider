Sci-fi UI HUD - Custom GUI HUD

This package is designed to make a custom gui HUD effects, this package includes the pre defined HUD prefabs and also includes the elements' png files which elables to make the custom animation effects.
If you want to change the color of any hud, select the child object that you want to change color of, and goto "Image" component inside inspector viewn and select color property to change color.
Every HUD has its own animator component and number of animations inside to corrosponding folder under animation folder.
THe scripts are defined for the circular motion of the HUD, and every HUD has own script.
All the prefabs are ready to use and demo scenes are also implemented.
Just drag the prefabs to use it into the scene.
Presentation scene has all the prefabs in full screen and in different colors, use them accourdingly.
